## 1. 채널 정체성

| 항목 | 내용 |
|---|---|
| **목적** | A(개인 브랜딩·리드 생성) 메인 + B(지식공유) 서브 |
| **포지셔닝** | 회계사 × 딜사이드(DD·밸류에이션·IM) × Claude Code 자동화 — 미국 유튜브에도 비어있는 교집합 |
| **제작 컨셉** | **"Faceless · AI-Visual + Real-Evidence"** — 얼굴 없음, 영상 비주얼은 AI 제작(경제명탐정 방식·촬영 없음), 단 **밸류에이션 산출물은 실제 엑셀/덱 캡쳐를 중간 삽입** |
| **음성** | 고급 AI 음성 내레이션 — **경제명탐정 톤(차분·권위형, 중속)**. 저급 TTS 금지(콘텐츠팜 신호 회피) |
| **길이** | 8~12분, 대본 기반 (벤치마크: Softgirlnocode 9.5분/14.5만뷰 공식) |
| **표준 구성** | 후킹 → 오늘 만들 것 → (개념) → **실제 산출물 데모(엑셀/덱 캡쳐 삽입)** → "한 단계 더" → 마무리/CTA |
| **영상 비주얼** | AI 제작(촬영 없음) — 경제명탐정 방식. 말하는 화면·개념·b-roll·전환 전부 AI. 데모 구간에만 실제 캡쳐를 컷인 |

## 2. 절대 규칙 (제작 가드레일)

1. **영상 비주얼은 AI 제작(촬영 없음).** 단 **밸류에이션 산출물 자체(Football Field·검증 시트·결론)는 실제 엑셀/덱 캡쳐를 컷인**한다. 증거(산출물)는 AI로 위조·재현하지 않는다 — 이게 증거우선 = 차별 자산. (AI가 채우는 건 말하는 화면·개념·b-roll·전환이고, 증거는 진짜 캡쳐다.)
2. **AI 활용 범위** = 전체 비주얼 베이스 + 인트로 타이포·개념 카드·b-roll·전환·썸네일·편집 보조. **단 데모 구간의 산출물 캡쳐는 예외(실물)**.
3. **기밀/NDA** = 실제 의뢰 건은 회사명 **중립 코드명** 마스킹 + 식별정보 노출 금지.
   - ✅ `Project-GreenPlus`는 **실제 의뢰 건이 아닌 공개자료(DART·웹서치) 기반 연습 케이스** → 실명·산출물 노출 무방.
   - ⚠️ 실제 의뢰 건(Jellyfish/Sungjin/RMK 등) 화면은 영상에 사용 금지.
4. **익명은 시작 전술** — 트래픽·신뢰 확보 후 단계적 공개로 리드 전환. 익명기에도 "실명만 가린 진짜 전문가" 페르소나 유지.
